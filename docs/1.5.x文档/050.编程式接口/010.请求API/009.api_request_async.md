Forest请求默认为同步的，可以通过`async()`方法设置为异步请求

> setAsync(boolean async) 设置是否异步
>- 参数`aysnc`: `true` 异步, `false` 同步
>
> sync() 设置为同步
>
> async() 设置为异步

```java
// 构建同步 Get 请求
Forest.get("/");
// 构建异步 Get 请求
Forest.get("/").async();
// 构建同步 Get 请求
Forest.get("/").sync();
```
